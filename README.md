## 📌 About

**Make UA** is an open-source Python tool developed under the **Mr-SxR** brand. It generates valid Facebook app User-Agents instantly. Simply run the script and press **Enter** each time to get a fresh, new User-Agent — as many times as you need.

---

## ▶️ How It Works

```
Run the script
    ↓
A Facebook User-Agent is displayed
    ↓
Press Enter → New User-Agent appears
    ↓
Repeat as many times as needed
```

---

## ⚙️ Installation

```bash
cd $HOME
rm -rf FB-APP_User-Agent_Maker
git clone --depth=1 https://github.com/Mr-SxR/FB-APP_User-Agent_Maker.git
cd FB-APP_User-Agent_Maker
git pull
python MAKE_UA.py
```
